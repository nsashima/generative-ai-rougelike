# 🧭 サシマのシレン

「サシマのシレン」は、自動生成されるダンジョンを探索するローグライクRPGです。  
地下10階の最深部に眠る伝説の「生成AIの秘宝」を手に入れ、凶悪なボス「邪教の心眼」を撃破してダンジョンを踏破しましょう！

---

## 🎮 ローカルでの起動方法

必要なパッケージをインストールし、開発用ローカルサーバーを起動します。

```bash
# 依存パッケージのインストール
npm install

# 開発用サーバーの起動
npm run dev
```

起動後、ブラウザで `http://localhost:5173`（またはターミナルに表示されるローカルIPアドレス）にアクセスしてください。

## ⌨️ 操作方法一覧

ゲームはキーボードおよび画面下部のモバイル用コントローラーでプレイできます。

| キー | 動作 |
| :--- | :--- |
| **`W` / `A` / `S` / `D`** または **`↑` / `←` / `↓` / `→`** | プレイヤーの移動・モンスターへの攻撃 |
| **`Space` (スペースキー)** | その場で1ターン待機（HPが少し回復） |
| **`>`** または **`.`** | 下り階段を進む（※B5Fはボス撃破後に階段が出現） |
| **`1` 〜 `0`** | インベントリのアイテムを使う / 装備する |
| **`Alt` + `1` 〜 `0`** | アイテムの詳細情報を調べる（モーダルが開きます） |
| **`Shift` + `1` 〜 `0`** | アイテムを足元に置く |
| **`M`** | サウンドのON/OFF切り替え |
| **`R`** | 最初からリトライ（リスタート） |
| **`H`** | ゲーム内ヘルプ画面を開く / 閉じる |

## 🚀 Render (render.com) へのデプロイ方法

Renderで本アプリケーションを公開する場合、次の2通りのデプロイ方法があります。  
**「静的サイト (Static Site)」としてのデプロイが最も簡単かつ無料で行えるため推奨されます。**

### 📋 方法1：Static Site (静的サイト) としてデプロイ【推奨・無料】
本アプリは完全にフロントエンドで動作するため、静的アセットとして公開するのが最適です。

1. [Render](https://dashboard.render.com/) にログインし、**「New +」** ➡ **「Static Site」** を選択します。
2. 対象のGitHub/GitLabリポジトリを連携・選択します。
3. 設定項目に以下を入力します：
   * **Build Command**: `npm run build`
   * **Publish Directory**: `dist`
4. **「Create Static Site」** をクリックします。数分でデプロイが完了し、公開URLが生成されます。
   *(※静的サイトの場合、サーバーを起動する「スタートコマンド」は不要です)*

---

### 📋 方法2：Web Service (Webサービス) としてデプロイ
何らかの理由で「Web Service」テンプレートでデプロイする必要がある場合は、以下の設定を使用してください。

1. Renderのダッシュボードから **「New +」** ➡ **「Web Service」** を選択します。
2. リポジトリを連携・選択します。
3. ランタイムに **「Node」** を選択し、設定項目に以下を入力します：
   * **Build Command**: `npm run build`
   * **Start Command**: `npm run start` (または `npx serve -s dist`)
4. **「Create Web Service」** をクリックしてデプロイを実行します。
