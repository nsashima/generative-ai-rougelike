# 🧭 サシマのシレン (Ver 1.2.0)

「サシマのシレン」は、自動生成されるダンジョンを探索するターン制ローグライクRPGです。  
地下20階の最深部に眠る伝説の「生成AIの秘宝」を手に入れるため、襲いかかるモンスターを撃破し、5階層ごとに待ち受ける強力なボスを倒してダンジョンを踏破しましょう！

---

## 🎮 ローカルでの起動方法

必要なパッケージをインストールし、開発用ローカルサーバーを起動します。

```bash
# 依存パッケージのインストール
npm install

# 開発用サーバーの起動
npm run dev
```

起動後、ブラウザで `http://localhost:5173`（またはターミナルに表示されるローカルIPアドレス）にアクセスしてください。

## ⌨️ 操作方法一覧

ゲームはキーボードおよび画面下部のモバイル用コントローラーでプレイできます。

### 基本操作
| キー | 動作 |
| :--- | :--- |
| **`W` / `A` / `S` / `D`** または **`↑` / `←` / `↓` / `→`** | プレイヤーの移動・モンスターへの攻撃 |
| **`Space` (スペースキー)** | その場で1ターン待機（HPが少し回復） |
| **`>`** または **`.`** | 下り階段を進む（※ボス階層はボス撃破後に階段が出現） |
| **`1` 〜 `0`** | インベントリのアイテムを使う / 装備する |
| **`Alt` + `1` 〜 `0`** | インベントリ内のアイテムの詳細情報を調べる |
| **`Shift` + `1` 〜 `0`** | インベントリ内のアイテムを足元に置く |
| **`M`** | サウンドのON/OFF切り替え |
| **`R`** | 最初からリトライ（リスタート） ※修飾キーなしの単体押しのみ有効 |
| **`H`** | ゲーム内ヘルプ画面を開く / 閉じる |

### 装備中のアイテム詳細表示ショートカット
現在装備しているアイテムのステータスや説明文を一発で確認できます。
* **`Alt` + `W`**：装備中の **武器 (Weapon)** の詳細を表示
* **`Alt` + `A`**：装備中の **防具 (Armor)** の詳細を表示
* **`Alt` + `R`**：装備中の **指輪 (Ring)** の詳細を表示
*(※対象のアイテムを装備していない場合は、ゲームログにメッセージが表示されます)*

## ⚔️ ダンジョン構造とボス階層

ダンジョンは5階層ごとにテーマとデザインが変わり、5の倍数フロアには強力なボスが待ち受けています。ボスを撃破すると、次の階層へ進むための階段が出現し、強力なユニーク装備をドロップします。

| 階層 | テーマ・デザイン | 登場ボス | 討伐ドロップアイテム |
| :--- | :--- | :--- | :--- |
| **1F 〜 5F** | 標準の洞窟 | **ゴブリンキング** (5F) | **キングの盾** (防具) |
| **6F 〜 10F** | 溶岩・魔界 | **邪教の心眼** (10F) | **心眼の指輪** (指輪) |
| **11F 〜 15F** | 氷結・氷室 | **結晶の守護者** (15F) | **吹雪の魔剣** (武器) |
| **16F 〜 20F** | 宇宙・虚無 | **深淵の覇王** (20F) | **生成AIの秘宝** (最終アイテム) |

## ⚖️ 特徴的なゲームシステム

### 1. 耐久値システムと商人（ショップ）
* 武器や防具には「耐久値」が存在し、攻撃したりダメージを受けたりすることで減少します。耐久値が0になるとその装備は消滅してしまいます。
* 耐久値は「修復の巻物」を使うことで回復できます。
* ダンジョン内に出現する「商人（M）」に話しかけると、ゴールドを使って買い物をしたり、不要なアイテムを売却できます。
* **売却リスト画面では、装備の横に `(耐久:現在値/最大値)` が表示され、状態を確認したうえで取引可能です。**

### 2. モンスターの動的スケーリング (深度強化)
* 各モンスターには「初めて登場する基準階層（例: ヴァンパイアなら7階）」が設定されています。
* それよりも深い階層（超過階層）で出現した個体は、深さに応じてステータスが自動的にスケーリング（強化）されます。
  * **超過1階層ごとに: HP +10%、攻撃力 +8%、防御力 +5%、獲得経験値(XP) +10%**
* これにより、同じ名前のモンスターでも深い階層ほど強くなり、プレイヤーの装備インフレに対して常に緊張感のある戦闘が楽しめます。

## 🛠️ 開発者用デバッグモード

開発および調整のためにデバッグモードが搭載されています。

1. ゲーム画面で **`Ctrl + Shift + D`** を同時に押します。
2. パスワード入力ダイアログが表示されたら、**`sashima`** と入力します。
3. 画面中央にデバッグパネルが出現し、以下の機能が実行可能になります：
   * **ボス階層へのワープ**: 地下5階・地下10階へ一瞬で移動
   * **無敵モード (God)**: HP 9999 / 攻撃力 999 に上昇
   * **マップ全開**: 未探索エリアを含むすべての壁や床、階段の位置を開示
   * **ゴールド追加**: 1000 G を即座に獲得
   * **全表示テスト部屋**: ゲーム内のすべてのアイテム（武器・防具・指輪・消費アイテム等）と、全てのモンスター（モブ各種・色違いバリエーション・ボス全て）が綺麗に整列した安全な展示用マップを自動生成します（この部屋の敵はAIが停止しており、攻撃や移動をしてきません）。

## 🚀 Render (render.com) へのデプロイ方法

### 📋 Static Site (静的サイト) としてデプロイ【推奨・無料】
本アプリは完全にフロントエンド（HTML/JS）で動作するため、静的サイトとしてのデプロイが最も簡単でおすすめです。

1. [Render](https://dashboard.render.com/) にログインし、**「New +」** ➡ **「Static Site」** を選択します。
2. 対象のGitHub/GitLabリポジトリを連携・選択します。
3. 設定項目に以下を入力します：
   * **Build Command**: `npm run build`
   * **Publish Directory**: `dist`
4. **「Create Static Site」** をクリックして完了です。数分で公開URLが生成されます。
